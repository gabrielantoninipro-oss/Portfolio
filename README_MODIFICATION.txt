PORTFOLIO GEII - GABRIEL ANTONINI

Contenu :
- index.html : accueil
- apropos.html : présentation
- competences.html : page compétences complète avec Concevoir et Vérifier
- projets.html : projets Thermomètre de Bain et Kart à Hélice
- contact.html : contact
- css/styles.css : design
- js/script.js : menu, recherche, accordéons et fenêtres modales
- images/ : logo, photo du kart et illustrations
- documents/ : dossiers DOCX des projets

Pour ouvrir :
1. Extraire le ZIP entièrement.
2. Ouvrir le dossier extrait.
3. Double-cliquer sur index.html.

Important :
Il ne faut pas ouvrir index.html directement dans le ZIP.
Il faut extraire tout le dossier pour que le CSS, les images et le JavaScript se chargent correctement.

Pour modifier :
- Texte : modifier les fichiers .html
- Couleurs / tailles : modifier css/styles.css
- Menu / interactions : modifier js/script.js
- Images : remplacer les fichiers dans images/
- Documents : remplacer ou ajouter des fichiers dans documents/
